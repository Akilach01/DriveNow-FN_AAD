# 🚗 DriveNow Car Rental System - Frontend

## 📖 Project Description
The **DriveNow Frontend** is a responsive web application that allows users to **register, log in, browse vehicles, and make bookings**.  
It communicates with the backend REST APIs to provide a seamless car rental experience.

---

## ✨ Features
- 👤 **User Registration & Login**  
- 🔐 JWT token storage & authentication  
- 🚘 Browse available vehicles  
- 📅 Book cars online  
- 📄 Manage profile & bookings  

---

## ⚙️ Technologies
- **HTML5, CSS3, Bootstrap**  
- **JavaScript (ES6), jQuery**  
- **AJAX for API calls**  
- **FontAwesome (UI icons)**  

---

## 🚀 Setup Instructions

### 1️⃣ Clone the Repository

git clone https://github.com/username/drivenow-frontend.git

📌 Backend
git clone https://github.com/username/drivenow-backend.git

2️⃣ Configure API Base URL

Edit app.js and set your backend URL:

const API_BASE_URL = "http://localhost:8080/api";

3️⃣ Run the Frontend

Simply open index.html in a browser.
